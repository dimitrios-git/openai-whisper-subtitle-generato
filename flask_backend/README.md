# Flask Backend

## Dev

Create an environment:

```bash
python3 -m venv myenv
```

Activate it:

```bash
source myenv/bin/activate
```

Install the required dependences:

```bash
pip install -r requirements.txt 
```

Start the backend:

```bash
python3 app
```

Stop the backend, by pressing `CTL+C`. Deactivate the environment:

```bash
deactivate
```
